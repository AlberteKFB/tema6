Grundlæggende Web

I tema 2 om grundlæggende web blev vi præsenteret for de helt basale funktioner i html og css så som grid og media queries. Udover det lærte vi også om forskellige desingkonventioner, styletiles, wireframes, tekstopsætning osv. Figma blev også en del af vores værktøjkasse under dette tema.

Opgaven i dette tema bestod i at kode vores første hjemmeside fra bunden i "mobile first", vi fik udleveret wireframes og layout digrammer, der skulle hjælpe os med at visualisere processen. i CSS fik vi lov at lege med styling af fonte, farve og billede opsætning ved hjælp af grid og flex.

Resultatet af min opgave kan findes via knappen.
